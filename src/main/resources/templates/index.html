Introduction

Ceci est une application permettant d'étudier les failles de sécurité les plus communes.

  • Injection SQL
  • XSS
  • Session fixation
  • XXE

Il s'agit d'un site de démonstration possédant des failles de sécurité, assurez vous de :

  • ne pas configurer le compte de base de donnée avec un accès administrateur
  • lancer le site avec uniquement localhost en écoute
