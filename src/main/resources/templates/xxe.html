XXE

L'objectif est d'utiliser une injection XXE afin de récupérer des données sensibles du serveur.

Essayer par exemple de récupérer le fichier :

  • Sous Windows : C:\Windows\System32\drivers\etc\hosts
  • Sous Linux : /etc/passwd

Pour celà, cette page permet d'importer des produits en base depuis un fichier XML

Vous pouvez télécharger le template de ce fichier ici : Template XML
